= 監視リストを作成します
:toc: macro
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toc: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/
:toc-position: content


[role="lead"]
一連のシステムにすばやく簡単にアクセスできる監視リストを作成できます。カテゴリまたは場所に基づいて、複数のシステムを組み合わせることができます。

.手順
. 左側のペインで、「 * Watchlists * 」をクリックします
+
[ 監視リストを管理 ] 画面が表示されます。

. 監視リストを作成 * をクリックします。
. 監視リストの名前を指定します。
. 監視リストに追加するシステムの詳細を指定します。
+
1 つのカテゴリに最大 15,000 台のシステムを追加でき、 1 つのシリアル番号には 500 台のシステムを追加できます。

. 監視リストを作成 * をクリックします。
+
監視リストダッシュボードが表示されます。
