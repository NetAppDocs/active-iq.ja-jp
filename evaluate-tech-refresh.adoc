= Digital Advisorでストレージシステムのシステム更新が推奨されるかどうかを判断します
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Tech Support契約またはハードウェアに対してシステム更新が推奨されるかどうかを確認したい場合は、システム更新オプションを使用できます。

この機能には、ライフサイクル計画（*ストレージ* > *ライフサイクル計画*）またはDigital Advisor（*ダッシュボード* > *プランニングウィジェット* > *システム更新*）からアクセスできます。

この機能の詳細については、コンソールのドキュメントのlink:https://docs.netapp.com/us-en/bluexp-economic-efficiency/use/tech-refresh.html["システムリフレッシュを評価する"]を参照してください。
